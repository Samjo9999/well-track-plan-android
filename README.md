# Well Track — Beta (Voice‑first Stimmungs‑ & Tagebuch‑App)

Kurzbeschreibung
- Voice‑first Stimmungs‑ und Tagebuch‑App zur Selbstbeobachtung (keine Diagnose).
- Frühe Beta, noch in Arbeit. App derzeit nur auf Deutsch.

Verteilung (wichtig)
- Die App wird über GitHub bereitgestellt.
- Lade die APK von der Release‑Seite dieses Repositories herunter:
  - https://github.com/Samjo9999/well-track-plan-android/releases
  - Dateiname: well-track-plan-beta-v0.1.apk
- Lade die APK nur von der offiziellen Release‑Seite herunter, damit du die echte Version bekommst.

Schnellstart — Installation auf Android
1. Öffne auf deinem Android‑Gerät die Release‑Seite und lade die APK herunter.
2. Erlaube bei Bedarf das Installieren aus unbekannten Quellen:
   - Beim Öffnen der APK wirst du ggf. zu Einstellungen → „Apps aus unbekannten Quellen installieren“ weitergeleitet. Erlaube die verwendete App (z. B. Browser, Files).
3. APK installieren und öffnen:
   - Tippe auf die heruntergeladene Datei → Installieren → Öffnen.
   - Wenn du möchtest, kannst du die Erlaubnis danach wieder entziehen.

Beta‑Status — was du erwarten solltest
- Android only (keine iOS‑Version).
- Nicht im Google Play Store — manuelles Installieren der APK erforderlich.
- Frühe Beta: UI‑Unschärfen, fehlende Features, mögliche Abstürze oder Transkriptionsfehler.
- Die App dient der Selbstbeobachtung; keine medizinische Diagnose oder Behandlung.

Feedback — bevorzugter Weg
- Bevorzugt: In‑App‑Feedback (Feedback‑Funktion in der App; derzeit auf Deutsch).
  - Das In‑App‑Feedback ist am hilfreichsten, da es automatisch Protokolle und ggf. Sprachnotizen anhängen kann.
- Falls In‑App nicht möglich:
  - Erstelle ein GitHub Issue und gib an:
    - Gerät und Android‑Version
    - Kurze Beschreibung + Schritte zur Reproduktion
    - Screenshots oder kurze Sprachaufnahmen, falls möglich
  - Alternativ: Nachricht an u/Samjo9999 auf Reddit.
- Kurze, klare Berichte helfen am meisten.

Sicherheit & Transparenz
- Keine medizinische Beratung: Die App ist nur für Selbstbeobachtung gedacht, nicht zur Diagnose oder Behandlung.
- Keine Datenverkäufe: Dieses Projekt verkauft keine Benutzerdaten.
- Frühphase: Datenschutz‑ und Funktionsumfang können sich noch ändern. Bei Fragen öffne bitte ein Issue.
- Wenn du entdeckst, dass sensible Daten extern gesendet werden, stoppe die Nutzung und melde es sofort.

Deinstallieren
- Standard Android: Einstellungen → Apps → App auswählen → Deinstallieren.
